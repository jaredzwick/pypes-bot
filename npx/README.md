# @pypes/bot

CLI for installing and running [pypes-bot](https://github.com/jaredzwick/pypes-bot) via Docker.

```bash
npx @pypes/bot init      # interactive setup, writes ./pypes.env
npx @pypes/bot start     # docker run with the env file mounted
```

See the [main README](https://github.com/jaredzwick/pypes-bot) for full setup instructions.
